# SmileSchool Homepage

Welcome to **SmileSchool** – a fictional learning platform designed to teach the art of smiling through engaging tutorials and expert guidance. This HTML page represents the homepage layout, built with semantic HTML5 elements.

---

## 📄 Project Overview

This project showcases a static web page layout using only HTML. It’s structured to highlight a landing page for an online tutorial platform focused on teaching various smile styles. The design includes key sections such as hero content, testimonials, featured instructors, video tutorials, membership perks, and FAQs.

---

## 🧩 Features & Sections

### ✅ Header
- Navigation bar with logo
- Links to "Courses", "Pricing", and "Login"

### 🎯 Banner (Hero Section)
- Eye-catching call-to-action: **"Get schooled"**
- Subtext words: *Smile*, *Grin*, *Laugh*
- Prominent **"Register For Free"** button
- Highlighted instructors with their awards and achievements

### 💬 Testimonials
- Quote from a user praising the platform
- Includes name and occupation

### 🎥 Most Popular Tutorials
- Four featured videos with:
  - Thumbnail images
  - Title and description
  - Tutor profile (Phillip Massey)
  - Star ratings
  - Duration of each video

### 🎁 Free Membership Section
- Three membership perks (placeholder content)
- Encouraging another **Register For Free** call-to-action

### ❓ FAQ Section
- Common questions with long-form answers
- Structured in two-column style for readability

### 🔗 Footer
- Logo and social media icons (Facebook, Twitter, Instagram)
- Copyright
